Introduction  
Cette application intègre les produits `Sure Petcare` dans Homey.  
Gérez l'état de vos chatières ou mangeoires et suivez vos animaux !

Utilisation  
1. Installez l'application  
2. Allez dans les paramètres de l'application  
3. Connectez-vous avec vos identifiants `Sure Petcare`.  
4. Ajoutez l'appareil à Homey.
